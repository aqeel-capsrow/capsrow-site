One map explains how Africa banked itself without branches.

For every 100,000 African adults there are about 340 mobile-money agents - and only about 6 bank branches. The counter is a person, not a building: a shopkeeper with a float and a phone, on the corner where you already are. Some 28 million agents now run this last mile, most of them added across sub-Saharan Africa.

When the branch never came, the neighbourhood became the branch.

Sources. Source: GSMA State of the Industry Report on Mobile Money, 2024.
